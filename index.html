Standalone single-stock tool. Runs the same Pivot Strategy engine as the main Backend's POST /backtest (Backend/backtest/algoStrategy.js), using the SBIN.NS-tuned parameters from shared/strategy-engine/stockConfigs.js. Table format matches the main StockPilot frontend.

Interval: 1-minute candles. Yahoo only serves 1-minute data for the trailing ~7 days, so the date pickers below are limited to that window.
